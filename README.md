# AI Document Summarizer + Q&A Web App

## Features
- Upload PDF, DOCX, TXT
- AI summarization
- Ask questions about document
- Chat-like interface

## Setup

1. Clone project
2. Install dependencies:

pip install -r requirements.txt

3. Add your Hugging Face API key in app.py:

HF_API_TOKEN = "your_token_here"

4. Run app:

python app.py

5. Open browser:

http://127.0.0.1:5000

## Notes
- Free API (Hugging Face)
- Works best with smaller documents (API limits)
